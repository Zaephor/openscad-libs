# Printing two-piece-box

| Setting | Recommendation |
|---|---|
| Material | PLA |
| Nozzle | 0.4 mm |
| Layer height | 0.2 mm |
| Walls / perimeters | 3 |
| Infill | 20% gyroid |
| Print speed | 50 mm/s |
| Supports | None |
| Orientation | Each part flat on bed (base open-side up, lid flat) |

Notes: Print body and lid separately. The body prints best with the open side facing up. The lid prints flat with no supports needed.
