# Printing cable-clip

| Setting | Recommendation |
|---|---|
| Material | PLA |
| Nozzle | 0.4 mm |
| Layer height | 0.2 mm |
| Walls / perimeters | 3 |
| Infill | 20% gyroid |
| Print speed | 50 mm/s |
| Supports | None |
| Orientation | Flat on bed (cable channel facing up) |

Notes: Print with the flat base on the bed. The cable channel and screw holes are self-supporting at 0.2 mm layer height.
